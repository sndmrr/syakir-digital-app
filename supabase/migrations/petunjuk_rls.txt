Petunjuk: Langkah-langkah penerapan RLS (copy-paste satu per satu)

Petunjuk singkat:
- Jalankan tiap blok SQL satu per satu di SQL editor Anda.
- Perintah `DROP POLICY IF EXISTS` akan menghapus kebijakan yang sudah ada sebelum membuat yang baru.
- Jika beberapa RLS sudah aktif, Anda bisa melewatkan langkah `ENABLE ROW LEVEL SECURITY` (menjalankannya aman jika sudah aktif).

1) Aktifkan RLS pada tabel `user_roles`:

ALTER TABLE public.user_roles ENABLE ROW LEVEL SECURITY;

2) Buat kebijakan: Users can view their own role

DROP POLICY IF EXISTS "Users can view their own role" ON public.user_roles;
CREATE POLICY "Users can view their own role"
ON public.user_roles FOR SELECT
USING (auth.uid() = user_id);

3) Buat kebijakan: Admins can view all roles

DROP POLICY IF EXISTS "Admins can view all roles" ON public.user_roles;
CREATE POLICY "Admins can view all roles"
ON public.user_roles FOR SELECT
USING (public.has_role(auth.uid(), 'admin'));

4) Buat kebijakan: Admins can insert roles

DROP POLICY IF EXISTS "Admins can insert roles" ON public.user_roles;
CREATE POLICY "Admins can insert roles"
ON public.user_roles FOR INSERT
WITH CHECK (public.has_role(auth.uid(), 'admin'));

5) Buat kebijakan: Admins can update roles

DROP POLICY IF EXISTS "Admins can update roles" ON public.user_roles;
CREATE POLICY "Admins can update roles"
ON public.user_roles FOR UPDATE
USING (public.has_role(auth.uid(), 'admin'));

6) Buat kebijakan: Admins can delete roles

DROP POLICY IF EXISTS "Admins can delete roles" ON public.user_roles;
CREATE POLICY "Admins can delete roles"
ON public.user_roles FOR DELETE
USING (public.has_role(auth.uid(), 'admin'));

USING (auth.uid() = user_id AND deleted_at IS NOT NULL);

Catatan tambahan:
- Pastikan fungsi `public.has_role(uid, role)` tersedia di database Anda.
- `auth.uid()` bergantung pada konteks Supabase; jalankan perintah ini di lingkungan Supabase SQL Editor.
- Jika Anda ingin memastikan kebijakan tertentu tidak diubah kecuali secara eksplisit, jalankan setiap blok dan verifikasi hasilnya sebelum melanjutkan.
